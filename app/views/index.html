Single User View

Internal facing single user view that displays multiple PCN's for single users. This should help identify repeat offenders or recurring issues with vulnerable people building up many large penalties.

Version 3 - February 2018

What we tested - We further tested functionality with dragable pop out windows that allow agents to see the record side by side with the function they are performing. We also briefly tested the possibility of having the multiple PCN's presented in this way. The purpose of this was to find the most effective way for users to perform daily functions and thus deal with cases quicker.

  • CCS unlock

Version 2 - January 2018

What we tested - Following the A/B test from v1, We tested the customer centered version with some dummy functionality links. We also tested a back office version of this that included a dummy button for reports and additional dummy options for transactions. The purpose of this testing was to discover more about the interactions that agents need to make with cases on a day to day basis.

  • CCS version
  • BO version

Version 1 - November 2017

What we tested - Version one of SUV journey was created to provide call center agents with a better view of patients that may have multiple PCN's. We tested two different layouts with 5 agents and 1 back office staff member. The matches and possible matches tabs received good feedback with agents appreciating the value of this feature. The customer centered view was the preferred layout.
